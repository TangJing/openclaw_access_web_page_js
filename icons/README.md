# 图标说明

当前目录包含 SVG 格式的图标文件。Chrome 扩展需要 PNG 格式的图标，你可以：

## 方法 1：使用在线工具转换
将 SVG 文件转换为 PNG 格式（16x16, 48x48, 128x128）

## 方法 2：使用 ImageMagick
```bash
convert icon16.svg icon16.png
convert icon48.svg icon48.png
convert icon128.svg icon128.png
```

## 方法 3：使用设计工具
- Figma
- Adobe Illustrator
- GIMP

## 图标尺寸要求
- **16x16**: 工具栏图标
- **48x48**: 扩展管理页面图标
- **128x128**: Chrome Web Store 图标
